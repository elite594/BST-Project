Binary Search Tree Implementation
==============================
Ayo Omotola  
id: 1331787
This program implements a Binary Search Tree (BST) that:
1. Generates 35 random numbers (1-99)
2. Creates nodes with left/right pointers
3. Inserts numbers into BST
4. Performs pre-order, in-order, and post-order traversals

Files:
- binarySTree.h: Class declarations
- binarySTree.cpp: Class implementations
- main.cpp: Main program
- readme.txt: This file

Output:
- Shows original random numbers
- Displays results of all three traversals
- In-order traversal shows numbers in ascending order
